CRIAR AMBIENTE DE DESENVOLVIMENTO PYTHON E ATIVÁ-LO

- Abra o terminal e digite python -m venv nome_do_ambiente virtual

- Ative o ambiente virtual digitando o seguinte comando dento do diretório do ambiente virtual python: Scripts/activate

CRIAR PROJETO DJANGO E RODÁ-LO

- Instale o django no ambiente virtual python, com o comando: pip install django

- No diretório que desejar, rodar o comando django-admin startproject nome_do_projeto

- Para testar, basta entrar no diretório do projeto e rodar o comando: python manage.py runserver

- Acessar pelo navegador o link: http://127.0.0.1:8000/



python manage.py migrate - Rodas as migrations para o banco sqlite
